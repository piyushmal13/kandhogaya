# Principles
- **Institutional Sovereignty**: Every UI element must feel premium, data-driven, and authoritative.
- **200x Evolution**: Adhere strictly to the design tokens in `index.css`.
- **Zero AI Slop**: No generic colors, no basic cards, no flat blocks.

# Before Coding:
1. **Direction**: Choose a bold aesthetic (Sovereign Glass / Architectural Dark / Neon Precision).
2. **Typography**: Define a clear hierarchy usingserif-sans pairings and mono-spaced utility markers.
3. **Interactions**: Define hover states using multi-layered shadows and subtle scale transforms.
4. **Accessibilty**: Ensure keyboard focus states are distinct and ARIA attributes are used for data-rich components.

# Execution:
- Produce production-ready, fully-typed React (`.tsx`) code.
- Use Tailwind CSS with the institutional color palette.
- Respect `prefers-reduced-motion` for all transitions.
- Ensure internal panel scrolling for long lists to maintain layout stability.
